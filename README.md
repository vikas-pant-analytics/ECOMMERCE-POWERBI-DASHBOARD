# E-commerce Sales Dashboard (Power BI)

## 📊 Project Overview

This project is an interactive Power BI dashboard built to analyze e-commerce sales performance.

## 🔧 Tools Used

* Power BI
* DAX
* Data Modeling

## 📁 Dataset

The dataset includes:

* Orders data
* Order details
* Sales target data

## 📈 Key Insights

* Total Sales: 432K
* Total Profit: 24K
* Total Orders: 1500
* Profit Percentage: 5.55%

## 📊 Dashboard Features

* KPI Cards for performance tracking
* Sales analysis by category
* Top customer identification
* State-wise sales visualization (Map)
* Interactive slicers for filtering

## 🎯 Skills Demonstrated

* Data modeling (relationships)
* DAX calculations
* Dashboard design
* Business analysis
